== Description ==

*Download RD Products* for free and start making money with amazon affiliates.

= #1 Easy to use =

With [RD Products](https://roberpena.com/plugins/no-api-amz/) you will get all the affiliate products you need in a few minutes. 
You only need your Amazon affiliate tag to start working.

= #2 Compatible with all the Markets =

At the config page, you can select the Amazon Market you want for search the products.
Download it now and start earning money with your affiliate products without the need of an API.
You could check if your server can connect to Amazon.

= #3 Fast and clean =

You will find a clean interface and really easy to use.
The plugin generates shortcodes for add them anywhere, posts, pages, widgets...

You have some options for customize the colors and button text.
